# AI Cover Letter & Cold Email Generator

Paste a job description + upload your CV → get a tailored cover letter or cold email in seconds.

**Live demo → [Try it here](https://meghnagupta1264.github.io/job-ai-tool/)**

## Features
- PDF CV upload (parsed directly by Claude AI)
- Paste any job description or URL
- Generates Cover Letters, Cold Emails, or both
- Real-time streaming output
- Tone selector (Professional, Confident, Concise...)

## Tech Stack
- Vanilla HTML/CSS/JS (zero dependencies, zero build step)
- Anthropic Claude API (claude-sonnet, streaming)
- PDF parsing via Claude's document API
- Hosted on GitHub Pages
